ACC (Adaptive Cruise Control)

Adaptive Cruise Control (ACC) Task Description The ACCTask is a FreeRTOS task that implements Adaptive Cruise Control (ACC), automatically adjusting the vehicle's speed based on the distance to obstacles detected by an ultrasonic sensor. The system maintains a safe following distance while optimizing speed for smooth operation. Key Functionalities: Sensor Data Acquisition: Safely reads filtered ultrasonic distance from a shared global structure (gFilteredData) using a mutex (xSensorDataMutex). If the mutex is unavailable, the task waits briefly (10ms) before retrying. Speed Control Logic: Obstacle Too Close (ultrasonic ≤ min_distance): Full stop (speed = 0) to prevent collision. Obstacle Within Adjustable Range (min_distance < ultrasonic < default_distance): Dynamically reduces speed based on distance (speed=max_speed - 2 × (default_distance - ultrasonic)). Ensures smooth deceleration as the vehicle approaches an obstacle. No Obstacle Detected (ultrasonic ≥ default_distance): Maintains maximum speed (speed=max_speed) for optimal travel.
