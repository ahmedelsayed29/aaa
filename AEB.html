AEB (Autonomous Emergency Braking)

Autonomous Emergency Braking (AEB) Task Description The AEBTask is a FreeRTOS task responsible for implementing Autonomous Emergency Braking (AEB) functionality in an embedded system. It continuously monitors sensor data (ultrasonic and infrared sensors) to detect obstacles or potential collisions. If an obstacle is detected within a critical distance, the system triggers an emergency stop, activates visual/audible alerts, and resumes normal operation once the path is clear. Key Functionalities: Sensor Data Acquisition: Safely reads filtered sensor data (ultrasonic distance and IR obstacle detection) from a shared global structure (gFilteredData) using a mutex (xSensorDataMutex). If the mutex is unavailable, the task waits briefly before retrying. Obstacle Detection & Emergency Braking: Triggers an emergency stop if: The ultrasonic sensor detects an obstacle within BREAK_DISTANCE. Both left and right IR sensors detect an obstacle (leftIR == 1 && rightIR == 1). Sends a STOP command to the motor control queue (xMotorQueue). Activates an LED (GPIOA PIN9) and Buzzer (GPIOA PIN4) for visual/audible warning. Sets the servo to a neutral position (90°)
