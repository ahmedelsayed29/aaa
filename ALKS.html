LKS (Lane Keeping System)

Lane Keeping System (LKS) Task Description The LKSTask is a FreeRTOS task responsible for implementing a Lane Keeping System (LKS) in an embedded autonomous vehicle or robotic system. It uses infrared (IR) sensors to detect lane markings and adjusts the vehicle's steering (via servo control) and speed to stay within the lane. Key Functionalities: Sensor Data Acquisition: Safely reads filtered left and right IR sensor data from a shared global structure (gFilteredData) using a mutex (xSensorDataMutex). If the mutex is unavailable, the task waits briefly (10ms) before retrying. Lane Detection & Steering Logic: Both IR sensors clear (leftIR == 0 && rightIR == 0): Maintains straight path (servo angle = 90°). Keeps speed at default (baseSpeed). Left IR detects line (leftIR == 1 && rightIR == 0): Reduces speed (if baseSpeed > 30, decreases by 20; otherwise, sets to 30). Turns left (servo angle = 75°) to correct deviation. Right IR detects line (leftIR == 0 && rightIR == 1): Reduces speed (same logic as left correction). Turns right (servo angle = 105°) to correct deviation. Both IR sensors detect lines (leftIR == 1 && rightIR == 1): Stops the vehicle (speed = 0). Resets servo to neutral (90°) (may indicate an intersection or obstacle).
