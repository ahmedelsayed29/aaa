FOTA (Firmware Over-The-Air)

Secure FOTA Command Bridge Our ESP32-based proxy enables secure, cloud-driven firmware updates for STM32 devices. It connects to an MQTT broker over TLS, validates incoming commands using CRC32, and forwards them via UART. Real-time feedback (ACK/NACK + data) is published back to the cloud, ensuring transparency and reliability. Designed with retry logic, timeout handling, and full buffer control for robust communication. This modular architecture keeps the bootloader secure and focused on flash operations only.
