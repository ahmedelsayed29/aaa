Wi-Fi Control

The Wi-Fi Control Task enables real-time wireless interaction with the vehicle using a UART-based Wi-Fi module (USART3). It continuously listens for character-based commands like 'f' for forward, 'b' for backward, 's' to stop, and 'l'/'r' for steering. Each command is interpreted instantly and translated into motor or servo control actions, ensuring smooth remote operation. Additionally, the task supports OTA (Over-The-Air) firmware updates when the 'u' command is received, triggering a system reset after flashing. Commands are processed with low latency (every 50ms), enabling a highly responsive control experience. The task integrates seamlessly with the motor and steering systems through message queues and hardware drivers. It offers a user-friendly and flexible way to control the vehicle without physical connection.
